# BB Visual Novel Engine

Расширение для SillyTavern, которое превращает обычный чат в более управляемую визуальную новеллу: с отношениями, памятью персонажей, вариантами действий и HUD-уведомлениями.

## Возможности

- **Social Link HUD** - боковая панель отношений с карточками персонажей.
- **Две шкалы отношений** - доверие и романтика считаются отдельно.
- **Карточки персонажей** - аватары, профили, базовые значения, статус и заметки для prompt.
- **Память персонажей** - мягкие следы, незабываемые события и архив важных моментов.
- **Кристаллизация черт** - глубокие воспоминания могут превращаться в постоянные черты характера.
- **Дневник и журнал** - история изменений, сюжетные моменты и системные записи.
- **Варианты действий** - три эмоционально размеченных ответа над полем ввода.
- **HUD-уведомления** - отдельные toast-стили для памяти, сближения, романтики, разлада и редких событий.

## Варианты действий

VNE может генерировать три действия для пользователя с разными намерениями и тоном. Каждый вариант содержит:

- `intent` - короткая суть действия;
- `tone` - эмоциональная окраска;
- `forecast` - краткий намёк на возможное последствие;
- `targets` - персонажи, которых действие затронет сильнее всего;
- `message` - готовый текст ответа.

Поддерживаются обычный reroll, умный reroll с пожеланием, авто-показ вариантов и длина ответа `short`, `medium` или `long`.

## Настройки и интеграция

- Можно включить или отключить авто-показ вариантов.
- Можно отключить трекер отношений, оставив генерацию вариантов.
- Можно настроить силу влияния событий отдельно для доверия и романтики.
- Поддерживается OpenAI-compatible Custom API для генерации вариантов и служебных запросов.
- Поддерживается ручная интеграция через макрос `{{bb_vn}}`.
- Snapshot-базу можно экспортировать, импортировать и очищать.

## Память и персонажи

- Данные привязаны к сообщениям, свайпам и активной персоне.
- Персонажей можно переименовывать, скрывать и восстанавливать.
- Для персонажа можно загрузить аватар, настроить кадрирование и сгенерировать профиль по шаблону.
- Платонический режим блокирует романтическую линию для выбранного персонажа.

## Установка

1. Откройте SillyTavern.
2. Перейдите в `Расширения -> Установить расширение`.
3. Вставьте ссылку на репозиторий:
   `https://github.com/maxkara14/BB-Visual-Novel-Engine`
4. Перезагрузите страницу.
5. Откройте настройки `BB Visual Novel Engine` и выберите нужные режимы автоматизации.
