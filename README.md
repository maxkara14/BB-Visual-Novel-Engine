**💖 BB Visual Novel Engine**

Расширение для SillyTavern, превращающий текстовую ролевую игру в полноценную визуальную новеллу.

Включает в себя динамическую систему отношений **Social Link** и панель интерактивного выбора действий.

**Что внутри?**

* **👥 Social Link HUD:** Боковая панель со списком персонажей. Отслеживает очки отношений (от -100 до +100) и визуализирует их через двусторонние прогресс-бары.
* **🎬 Интерактивное кино:** Специальная панель действий над полем ввода. В один клик ИИ генерирует 3 варианта хода с разным уровнем риска. Выбираете вариант — и расширение само пишет и отправляет пост от лица вашего персонажа.
* **📜 Журнал:** Отдельная вкладка с логом всех событий: знакомства, изменения симпатии и системные уведомления.

**Особенности:**

* **🧠 Статусы от ИИ:** Нейросеть сама придумывает уникальные статусы («Тайная симпатия», «Надежный напарник», «Заклятый соперник») исходя из контекста вашей истории.
* **⚠️ Механика Риска:** Выбор рискованных действий напрямую влияет на характер генерации ответа.
* **🔄 Умная синхронизация:** Статы отношений привязаны к чату. Если вы удалите сообщение или свайпнете ответ — отношения мгновенно пересчитаются.

---

**⚙️ Установка:**

1. Откройте SillyTavern.
2. Перейдите во вкладку *Расширения -> Установить расширение*.
3. Вставьте ссылку на этот GitHub репозиторий: `https://github.com/maxkara14/BB-Visual-Novel-Engine`
4. Перезагрузите страницу (F5).
5. Настройте авто-отправку и генерацию во вкладке Расширения.
