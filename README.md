**💖 BB Visual Novel Engine (VGE)**

Расширение для SillyTavern, превращающее текстовую ролевую игру в полноценную визуальную новеллу.

Включает в себя динамическую систему отношений **Social Link** и панель интерактивного выбора действий с поддержкой молниеносной генерации.

**Что внутри?**

* **👥 Social Link HUD:** Боковая панель со списком персонажей. Отслеживает очки отношений (от -100 до +100) и визуализирует их через двусторонние прогресс-бары.
* **🎬 Интерактивное кино:** Специальная панель действий над полем ввода. В один клик ИИ генерирует 3 варианта хода с разным уровнем риска.
* **👁️ Режим предпросмотра:** Тексты ответов генерируются заранее. Кликните на вариант и готовый текст появится в поле ввода чата. Вы можете прочитать его, отредактировать или выбрать другую кнопку перед отправкой.
* **⚡ Кастомный API:** Возможность подключить отдельный API-ключ специально для генерации ответов. Генерируйте варианты быстрей, не тратя лимиты вашей основной модели.
* **📜 Журнал:** Отдельная вкладка с логом всех событий: знакомства, изменения симпатии и системные уведомления.

**Особенности:**

* **⚠️ Механика Риска:** Выбор рискованных действий напрямую влияет на характер генерации ответа.
* **🔄 Умная синхронизация:** Статы отношений привязаны к чату. Если вы удалите сообщение или свайпнете ответ — отношения будут пересчитаны.
* **📱 Адаптивный дизайн:** Интерфейс кнопок и панелей полностью адаптирован под мобильные устройства и узкие экраны.
* **⚙️ Макросы:** Поддержка макроса `{{bb_vn}}` для ручного внедрения правил в ваши кастомные пресеты.

---

**🛠 Установка:**

1. Откройте SillyTavern.
2. Перейдите во вкладку *Расширения -> Установить расширение*.
3. Вставьте ссылку на этот GitHub репозиторий: `https://github.com/maxkara14/BB-Visual-Novel-Engine`
4. Перезагрузите страницу (F5).
5. Настройте авто-отправку и подключите API во вкладке Расширения.
